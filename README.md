# Спряжение ивритских глаголов

Интерактивные таблицы спряжения 135 глаголов из словаря учебника **«Иврит с нуля. Часть 1»** (Шломит Хаят, Сара Исраэли, Хила Кобринер).

🔗 **Сайт:** https://faxyua.github.io/verbs-hebrew/

## Что внутри

- **135 глаголов** в пяти биньянах:
  - פָּעַל / קל — 79
  - פִּעֵל — 21
  - הִפְעִיל — 20
  - הִתְפַּעֵל — 11
  - נִפְעַל — 4
- **Три времени** для каждого глагола:
  - Настоящее (4 формы)
  - Прошедшее (9 форм)
  - Будущее (8 форм)
- Полный никуд, ударный слог выделен красным.

## Поиск

Один общий поиск работает и по русскому, и по ивриту:

- **По переводу:** введите русское слово (полное или часть) — `писать`, `помнит`, `смех`.
- **По ивриту без никуда:** `כתב`, `אמר`, `סדר` — найдёт все формы, в основах которых эти буквы встречаются.
- **По ивриту с никудом:** `לִכְתּוֹב`, `מְדַבֵּר` — точное совпадение тоже работает.
- **Горячие клавиши:** `/` — фокус на поиск, `Esc` — очистить.

## Развёртывание на GitHub Pages

Этот репозиторий настроен как обычный сайт. Файл `index.html` — единственное, что нужно для работы (всё инлайн: стили, скрипт, данные).

1. Создайте репозиторий `verbs-hebrew` под аккаунтом `faxyua` (или используйте уже существующий).
2. Положите в корень репозитория файл **`index.html`** (из этого пакета).
3. В настройках репозитория откройте **Settings → Pages** и включите GitHub Pages:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (или `master`), папка `/ (root)`
4. Через 1–2 минуты сайт будет доступен по адресу:
   ```
   https://faxyua.github.io/verbs-hebrew/
   ```

## Печать

В браузере **Ctrl/Cmd + P** → формат A4, альбомная ориентация. Шапка поиска и подсказки автоматически скрываются при печати; разрывы страниц расставлены между биньянами и широкими таблицами.
